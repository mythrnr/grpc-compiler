# protobuf-compiler

[English](./README.md)

gRPCのコードをDockerで手軽に生成するツール。  
プロジェクトごとに用意するのが面倒だったのでツール化。

## 使い方

- 詳細は [Examples](./examples) を参照
- [docker-compose.yml](./examples/docker-compose.yml) に
`.proto` ファイルのディレクトリと出力先を指定してコンパイル

```bash
docker-compose up
```
